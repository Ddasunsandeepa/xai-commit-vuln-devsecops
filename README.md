# XAI Commit-Level Vulnerability Intelligence for DevSecOps

This repository contains the implementation of a final year research project focused on:

- Commit-level vulnerability prediction  
- Explainable AI for software security  
- DevSecOps CI/CD pipeline integration  

## Technologies
- Python  
- PyDriller  
- Machine Learning / CodeBERT  
- Explainable AI  
- Docker  

## Setup

Run using Docker:

docker-compose build
docker-compose run fyp
